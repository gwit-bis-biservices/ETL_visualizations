# GWU IDMC ETL Visualizations

A collection of internal web-based tools to explore and understand the ETL infrastructure across environments.

## Visualizations

- **Asset Explorer** — Browse all IDMC assets (mappings, taskflows, connections) across dev, QA, and prod.
- **GW Pipeline Explorer** — Visualize the SQL logic behind each mapping as a step-by-step flow, with environment-by-environment comparisons.
- **Lineage Visualization** — Trace data end-to-end from source systems through every transformation to its final target.
- **Tableau Lineage Visualization** — Connect Tableau dashboards and data sources back to their ETL pipeline origins.

## Usage

Hosted via GitHub Pages. Open `index.html` to access the main navigation page.
